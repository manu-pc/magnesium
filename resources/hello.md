# Welcome to Magnesium!

## What is this?
Magnesium is a **Markdown** editor with live preview and customizable styles. It's intended for clean note-taking, code documentation and report writing.

1. You have all the typical Markdown stuff like **lists** *(try adding to this one)!*
```
Or multi-line
     codeblocks!
```

## But with customization!
> So by clicking the gear ([gear]) icon on top, you can individually change the format (text color, fonts, sizes...) of each marker!

+ You can do stuff like create custom list types
- Like a "pros and cons" list

* You can also quickly type special saved characters like [pi] or smirkcat!
